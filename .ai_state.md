# Project Objective: Sanskrit Evidence Atlas

<!-- NB: local ../DCS/ export is reference data (text list + bibliography + 78k word→grammar list), NOT a passage corpus. See docs/DCS_SCHEMA.md. -->


Build an interactive, documentation-first research atlas connecting corpus evidence, grammar, and Sanskrit lexicography, where every claim is traceable `Claim -> Evidence -> Source`. Static-first (Observable Framework). Phase order: docs (Phase 0) -> MW Quantitative Depth (Phase 1) -> Comparative Dictionary Lab (Phase 2) -> DCS corpus grammar (Phase 3). See `ARCHITECTURE.md`.
