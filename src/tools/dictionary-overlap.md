# Pairwise dictionary overlap

How much vocabulary each pair of dictionaries shares, by normalized lemma. Jaccard = shared / union.

```js
const data = FileAttachment("../data/dicts/pairwise-overlap.json").json();
```

```js
import * as Plot from "npm:@observablehq/plot";
```

```js
const dicts = data.dictionaries.map(d => d.label);
// symmetric cells for a heatmap
const cells = [];
for (const p of data.pairwise) {
  cells.push({ a: p.a, b: p.b, shared: p.shared, jaccard: p.jaccard });
  cells.push({ a: p.b, b: p.a, shared: p.shared, jaccard: p.jaccard });
}
display(Plot.plot({
  marginLeft: 60,
  marginBottom: 50,
  width: 560,
  height: 480,
  color: { scheme: "blues", label: "Jaccard", legend: true },
  x: { domain: dicts, label: null },
  y: { domain: dicts, label: null },
  marks: [
    Plot.cell(cells, { x: "a", y: "b", fill: "jaccard" }),
    Plot.text(cells, { x: "a", y: "b", text: d => d.jaccard.toFixed(2), fill: d => d.jaccard > 0.4 ? "white" : "black", fontSize: 11 })
  ]
}));
```

## Pairwise table

```js
display(Inputs.table(data.pairwise.map(p => ({ "dict A": p.a, "dict B": p.b, "shared lemmas": p.shared, jaccard: p.jaccard })), {
  columns: ["dict A", "dict B", "shared lemmas", "jaccard"],
  sort: "shared lemmas",
  reverse: true
}));
```

<div class="note">The Petersburg pair (PWG / PWK) and MW / PWK show the strongest overlap, reflecting their shared lexicographic lineage. The Sanskrit-Sanskrit lexica (VCP, SKD) overlap less with the bilingual dictionaries.</div>
