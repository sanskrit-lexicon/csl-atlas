# DCS corpus inventory

The texts and word-grammar profile of the [Digital Corpus of Sanskrit](http://kjc-fs-cluster.kjc.uni-heidelberg.de/dcs/) reference export (UC-CG-01).

```js
const dcs = FileAttachment("../data/corpus/dcs-manifest.json").json();
```

```js
import * as Plot from "npm:@observablehq/plot";
```

<div class="warning" label="Reference data, not a passage corpus">
The local DCS export is <b>reference data</b> — a text inventory, a bibliography, and a word → grammar-category list — <b>not</b> passage-level occurrences. So lemma-frequency-by-text, POS-by-genre, and inflection profiles are <b>not</b> available here; they need the full DCS corpus. See <a href="https://github.com/sanskrit-lexicon/csl-atlas/blob/main/docs/DCS_SCHEMA.md">docs/DCS_SCHEMA.md</a>. Also note ${dcs.wordGrammar.lossyWords.toLocaleString()} of ${dcs.wordGrammar.totalWords.toLocaleString()} CSV words carry a <code>?</code> diacritic-loss artifact.
</div>

```js
display(html`<div style="display:flex;gap:24px;flex-wrap:wrap;margin:8px 0">
  ${[["Texts", dcs.textCount], ["Words (grammar list)", dcs.wordGrammar.totalWords], ["Grammar categories", dcs.wordGrammar.categories.length]].map(([l, v]) => html`<div>
    <div style="font-size:1.7rem;font-weight:700">${v.toLocaleString()}</div>
    <div style="color:var(--theme-foreground-muted);font-size:.85rem">${l}</div></div>`)}
</div>`);
```

## Word grammar profile

Aggregate distribution of the DCS word list by grammar group (no passages or frequency).

```js
const groups = Object.entries(dcs.wordGrammar.byGroup).map(([group, count]) => ({ group, count }));
display(Plot.plot({
  marginLeft: 110,
  height: 240,
  x: { label: "words", grid: true },
  y: { label: null, sort: { y: "x", reverse: true } },
  marks: [Plot.barX(groups, { x: "count", y: "group", fill: "var(--theme-foreground-focus)" }), Plot.ruleX([0])]
}));
```

```js
display(Inputs.table(dcs.wordGrammar.categories.map(c => ({ category: c.gram, group: c.group, words: c.count })), {
  columns: ["category", "group", "words"], sort: "words", reverse: true, rows: 12
}));
```

## Texts

```js
const tq = view(Inputs.text({ label: "Filter", placeholder: "title or code", submit: false }));
```

```js
const q = (tq ?? "").trim().toLowerCase();
const rows = dcs.texts.filter(t => !q || t.title.toLowerCase().includes(q) || t.code.toLowerCase().includes(q));
display(html`<p style="color:var(--theme-foreground-muted)">${rows.length} of ${dcs.textCount} texts${dcs.textsEnrichedFromBibliography ? ` · ${dcs.textsEnrichedFromBibliography} with bibliographic detail` : ""}</p>`);
display(Inputs.table(rows.map(t => ({ code: t.code, title: t.title, bibliography: t.citation ?? "" })), {
  columns: ["code", "title", "bibliography"],
  width: { code: 80, title: 240 }
}));
```
