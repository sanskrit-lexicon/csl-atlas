# Review queue · gender conflicts

Cross-dictionary gender disagreements among the tagged dictionaries (MW, AP, PWG, PWK, WIL), as review reports. Each is a candidate for human judgment — follow the source links and decide.

```js
const review = FileAttachment("../data/review/gender-conflicts-review.json").json();
```

<div class="warning" label="Evidence: derived · review overlay">
Gender is <b>derived</b> from <code>&lt;lex&gt;</code> tags; a conflict means two dictionaries assert disjoint specific genders. This page reads the review overlay (<code>src/data/review/gender-conflicts-review.json</code>, schema <code>data/schema/review-report.schema.json</code>). To record a decision, set an item's <code>reviewStatus</code> / <code>reviewedValue</code> / <code>reviewer</code> in that file; rebuilds preserve it by <code>reviewId</code>.
</div>

```js
const statusCounts = {};
for (const it of review.items) statusCounts[it.reviewStatus] = (statusCounts[it.reviewStatus] || 0) + 1;
display(html`<div style="display:flex;gap:24px;flex-wrap:wrap;margin:8px 0">
  ${[["Total", review.items.length], ...Object.entries(statusCounts)].map(([l, v]) => html`<div>
    <div style="font-size:1.6rem;font-weight:700">${v.toLocaleString()}</div>
    <div style="color:var(--theme-foreground-muted);font-size:.85rem">${l}</div></div>`)}
</div>`);
```

```js
const statusFilter = view(Inputs.select(["(all)", ...Object.keys(statusCounts)], { label: "Status" }));
const lemmaFilter = view(Inputs.text({ label: "Lemma contains", placeholder: "SLP1", submit: false }));
```

```js
const q = (lemmaFilter ?? "").trim();
const filtered = review.items.filter(it =>
  (statusFilter === "(all)" || it.reviewStatus === statusFilter) &&
  (!q || it.subject.lemma.includes(q))
);
const shown = filtered.slice(0, 250);
display(html`<p style="color:var(--theme-foreground-muted)">${filtered.length.toLocaleString()} item${filtered.length === 1 ? "" : "s"}${filtered.length > shown.length ? ` · showing first ${shown.length}` : ""}</p>`);
```

display(Inputs.table(shown.map(it => ({
