# Gender disagreements across dictionaries

Lemmas where two tagged dictionaries assert **disjoint** specific genders ({m, f, n}). These are review candidates, not verdicts — follow the source links to judge each case.

```js
const pos = FileAttachment("../data/dicts/pos-disagreement.json").json();
const conf = FileAttachment("../data/dicts/alignment-confidence.json").json();
```

<div class="warning" label="Evidence: derived">
Gender is <b>derived</b> from <code>&lt;lex&gt;</code> tags in the grammar-reliable dictionaries (MW, AP, PWG, PWK, WIL). VCP and SKD encode gender in prose and are excluded from conflict detection in this slice. A conflict means the dictionaries disagree on a specific gender; adjective/indeclinable tags never trigger one.
</div>

```js
display(html`<p><b>${pos.conflictCount.toLocaleString()}</b> lemmas show a gender disagreement; showing a sample of <b>${pos.shown}</b>.</p>`);
```

```js
display(Inputs.table(pos.conflicts.map(c => ({
  lemma: c.lemma,
  ...Object.fromEntries(Object.entries(c.byDict).map(([d, g]) => [d, g.join("/")])),
  sources: c.examples
})), {
  columns: ["lemma", "MW", "AP", "PWG", "PWK", "WIL", "sources"],
  format: {
    sources: examples => html`${examples.map(e => html`<a href=${e.href} target="_blank" rel="noopener" style="margin-right:6px">${e.dict}</a>`)}`
  }
}));
```

## Alignment confidence

How confidently lemmas align across dictionaries.

```js
display(html`<ul>
  <li><b>high</b> — every dictionary used the identical raw headword: ${conf.distribution.high.toLocaleString()}</li>
  <li><b>medium</b> — matched only after normalization (accent/digit stripping): ${conf.distribution.medium.toLocaleString()}</li>
</ul>`);
```

<div class="note">Because all seven dictionaries store SLP1 headwords, cross-dictionary alignment is almost always byte-identical — alignment uncertainty is low. The harder, deferred cases are homonym splitting and sense alignment (see the comparison plan).</div>
