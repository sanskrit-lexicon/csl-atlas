# MW lexical-family depth

The deepest lexical families in MW, ranked by a composite depth score. Families are grouped by the shared leading base of `<k2>`.

```js
const data = FileAttachment("../data/mw/mw-deepest-families.json").json();
```

```js
import * as Plot from "npm:@observablehq/plot";
```

<div class="warning" label="Evidence: inferred">
Family grouping is <b>inferred</b> — MW has no explicit derivation link, so families are approximated by shared <code>&lt;k2&gt;</code> base. Verify any family against its source records via the links. <code>depthScore</code> is for ranking only; read the components.
</div>

```js
display(html`<p><b>${data.familyCount.toLocaleString()}</b> families with more than one member; showing the deepest <b>${data.shown}</b>.</p>`);
```

## Depth vs breadth

```js
display(Plot.plot({
  height: 420,
  grid: true,
  x: { label: "members (breadth) →", type: "log" },
  y: { label: "↑ max compound depth" },
  marks: [
    Plot.dot(data.families, {
      x: "memberCount",
      y: "maxCompoundDepth",
      r: "sourceLayerSpan",
      fill: "earliestLayer",
      title: d => `${d.familyKey}\nmembers ${d.memberCount}, depth ${d.maxCompoundDepth}, span ${d.sourceLayerSpan}\n${d.earliestLayer} → ${d.latestLayer}`,
      opacity: 0.7
    })
  ]
}));
```

## Deepest families

```js
display(Inputs.table(data.families.map(f => ({
  family: f.familyKey,
  members: f.memberCount,
  depth: f.maxCompoundDepth,
  span: f.sourceLayerSpan,
  layers: `${f.earliestLayer ?? "?"} → ${f.latestLayer ?? "?"}`,
  citations: f.citationCount,
  score: f.depthScore,
  deepest: f.topExamples[0]?.href ?? ""
})), {
  columns: ["family", "members", "depth", "span", "layers", "citations", "score", "deepest"],
  format: { deepest: href => href ? html`<a href=${href} target="_blank" rel="noopener">record</a>` : "" },
  sort: "score",
  reverse: true
}));
```
