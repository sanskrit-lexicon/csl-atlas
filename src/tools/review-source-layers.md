# Review queue · unknown source layers

MW `<ls>` source abbreviations that don't yet map to a diachronic layer (UC-DIA-07), cited at least 5 times. Classify each into a layer to improve the diachronic profile.

```js
const review = FileAttachment("../data/review/unknown-source-layers-review.json").json();
```

<div class="warning" label="Evidence: derived · review overlay">
These sigla resolve to <code>unknown</code> after editorial-reference and base-form fallback. To resolve an item, add the source to <code>src/data/mw-source-layers.json</code> with its layer — it then leaves this queue and the MW diachronic profile improves. To record the decision here, set the item's <code>reviewStatus</code>/<code>reviewedValue</code> (e.g. <code>{ "layer": "vedic" }</code>)/<code>reviewer</code> in <code>src/data/review/unknown-source-layers-review.json</code>; rebuilds preserve it by <code>reviewId</code>.
</div>

```js
const statusCounts = {};
for (const it of review.items) statusCounts[it.reviewStatus] = (statusCounts[it.reviewStatus] || 0) + 1;
display(html`<div style="display:flex;gap:24px;flex-wrap:wrap;margin:8px 0">
  ${[["In queue", review.items.length], ["Distinct unknown (all)", review.distinctUnknownSources], ...Object.entries(statusCounts)].map(([l, v]) => html`<div>
    <div style="font-size:1.6rem;font-weight:700">${v.toLocaleString()}</div>
    <div style="color:var(--theme-foreground-muted);font-size:.85rem">${l}</div></div>`)}
</div>`);
```

```js
const statusFilter = view(Inputs.select(["(all)", ...Object.keys(statusCounts)], { label: "Status" }));
const srcFilter = view(Inputs.text({ label: "Source contains", placeholder: "e.g. Up, Br", submit: false }));
```

```js
const q = (srcFilter ?? "").trim();
const filtered = review.items.filter(it =>
  (statusFilter === "(all)" || it.reviewStatus === statusFilter) &&
  (!q || it.machineValue.source.includes(q))
);
const shown = filtered.slice(0, 250);
display(html`<p style="color:var(--theme-foreground-muted)">${filtered.length.toLocaleString()} item${filtered.length === 1 ? "" : "s"}${filtered.length > shown.length ? ` · showing first ${shown.length}` : ""}</p>`);
```

```js
display(Inputs.table(shown.map(it => ({
  source: it.machineValue.source,
  citations: it.machineValue.frequency,
  status: it.reviewStatus,
  proposedLayer: it.reviewedValue?.layer ?? "",
  example: it.sourcePointers[0]?.href ?? ""
})), {
  columns: ["source", "citations", "status", "proposedLayer", "example"],
  format: { example: href => href ? html`<a href=${href} target="_blank" rel="noopener">MW record</a>` : "" },
  sort: "citations",
  reverse: true
}));
```
