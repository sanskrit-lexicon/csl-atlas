# Review queue · source-siglum aliases

Source sigla cited by only **one** of the tagged dictionaries (MW, AP, PWG, PWK). Each may be genuinely unique to that dictionary, or the same source another dictionary cites under a different abbreviation — in which case it should be aliased so cross-dictionary citation comparison aligns it.

```js
const review = FileAttachment("../data/review/source-siglum-review.json").json();
```

<div class="warning" label="Evidence: derived · review overlay">
Candidates are canonical sigla cited only by one tagged dictionary, at least ${review.minCitations} times, after the diacritic/case fold. To merge a siglum with another dictionary's, add an alias to <code>src/data/dict-source-aliases.json</code> — it then aligns and leaves this queue. To record the decision here, set <code>reviewStatus</code>/<code>reviewedValue</code> (e.g. <code>{ "canonical": "p" }</code>)/<code>reviewer</code> in <code>src/data/review/source-siglum-review.json</code>; rebuilds preserve it by <code>reviewId</code>.
</div>

```js
const statusCounts = {};
for (const it of review.items) statusCounts[it.reviewStatus] = (statusCounts[it.reviewStatus] || 0) + 1;
display(html`<div style="display:flex;gap:24px;flex-wrap:wrap;margin:8px 0">
  ${[["In queue", review.items.length], ...Object.entries(statusCounts)].map(([l, v]) => html`<div>
    <div style="font-size:1.6rem;font-weight:700">${v.toLocaleString()}</div>
    <div style="color:var(--theme-foreground-muted);font-size:.85rem">${l}</div></div>`)}
</div>`);
```

```js
const dictFilter = view(Inputs.select(["(all)", ...new Set(review.items.map(i => i.machineValue.citedOnlyBy))], { label: "Cited only by" }));
const srcFilter = view(Inputs.text({ label: "Siglum contains", submit: false }));
```

```js
const q = (srcFilter ?? "").trim();
const rows = review.items.filter(it =>
  (dictFilter === "(all)" || it.machineValue.citedOnlyBy === dictFilter) &&
  (!q || it.machineValue.siglum.includes(q))
);
display(html`<p style="color:var(--theme-foreground-muted)">${rows.length} of ${review.items.length}</p>`);
display(Inputs.table(rows.map(it => ({
  siglum: it.machineValue.siglum,
  "cited only by": it.machineValue.citedOnlyBy,
  citations: it.machineValue.citations,
  status: it.reviewStatus,
  example: it.sourcePointers[0]?.href ?? ""
})), {
  columns: ["siglum", "cited only by", "citations", "status", "example"],
  format: { example: href => href ? html`<a href=${href} target="_blank" rel="noopener">record</a>` : "" },
  sort: "citations",
  reverse: true
}));
```
